# Lexi

> A macOS app for translating selected text using various LLM engines.

## Features

- Floating translation popup with a sleek, glass-like design
- Global hotkey support for quick access
- Integration with multiple LLM APIs including OpenAI and DeepSeek
- Streamed translation results for real-time feedback
- Markdown rendering for formatted translation output
- Easy-to-use settings for API keys and model selection
