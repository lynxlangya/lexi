选中文字，一键翻译。

Lexi 是一款 macOS 14+ 极简划词翻译浮窗：全局快捷键唤起，OpenAI 兼容流式翻译（OpenAI / DeepSeek），并为单词学习提供词典卡片（US 音标 / 释义 / 朗读）。

功能亮点

极简体验，但细节到位

划词即用

在任意 App 选中文字，按快捷键立即显示翻译浮窗，无需切换上下文。

毛玻璃浮窗

原生 Material 风格，靠近鼠标显示；点击外部自动关闭并清空内容。

流式输出

OpenAI 兼容 SSE 流式解析，边生成边展示（OpenAI / DeepSeek）。

单词学习卡片

选中单词时自动生成词典样式解释（US 音标 + 词性释义），支持朗读。

常驻后台

菜单栏入口 + 全局快捷键，启动后保持可用，不打扰但随时可用。

可配置

源/目标语种、快捷键、开机自启、引擎切换、自定义 OpenAI 兼容引擎。

如何使用

1 分钟上手

安装

下载 DMG 后，把 Lexi.app 拖入 /Applications。

  • 1 首次打开如被拦截：右键 app → Open，或在 System Settings → Privacy & Security 允许打开。
  • 2 在任意 App 选中文字，按 ⌘⇧L 唤起浮窗。
  • 3 打开设置填写 API Key / Base URL，或使用内置 Google 引擎。

权限与隐私

Lexi 需要 Accessibility 权限读取选中文本。API Key 存储在 macOS Keychain。

  • A System Settings → Privacy & Security → Accessibility → 启用 Lexi。
  • B 选中文本会发送给你选择的翻译服务（OpenAI / DeepSeek / Google）。请避免翻译敏感内容。

提示：如果刚开启 Accessibility，重启 Lexi 一次即可生效。

翻译引擎

内置 + 可扩展

Google（免费）

无需 Key，适合轻量翻译（可能被限流）。

gpt-4o（流式）

OpenAI 兼容接口，支持 SSE 流式输出，体验更顺滑。

deepseek-chat（流式）

OpenAI 兼容接口。把 Base URL 设置为 DeepSeek，直接使用。

Base URL 示例

OpenAI：https://api.openai.com/v1
DeepSeek：https://api.deepseek.com （Lexi 会自动补全 /v1/chat/completions）

你也可以在 Settings 添加任意 OpenAI 兼容引擎（自定义 Base URL + model + key）。
