Quick Start

三步上手

第一次使用只需要几分钟，之后就可以随时划词翻译。

  • 1
    下载并拖入 /Applications
    首次打开若被拦截，请在 Finder 中右键点击 Open。
  • 2
    授予辅助功能权限
    系统设置 → 隐私与安全性 → 辅助功能，开启 Lexi。
  • 3
    打开设置并选择引擎
    填写 API Key 或直接使用 Google 免费引擎体验。
