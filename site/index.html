Features

不仅是翻译，更是阅读助手

极致速度

选中文字，按下快捷键，0.3s 内响应。无需切换应用，保持当前阅读心流。

AI 流式输出

兼容 OpenAI 接口协议。支持 DeepSeek、GPT-4o，像真人一样逐字吐出翻译结果。

单词卡片

智能识别单词模式，自动展示美式音标、词性、释义。点击单词即可发音。

隐形存在

点击外部自动消失，没有任何多余的窗口残留。用完即走，不打扰你的工作区。

高度可配置

自定义 Base URL (方便接入 One API 或本地 LLM)、自定义 Model Name。内置 Google 翻译兜底，没有 Key 也能用。

Quick Start

即刻上手

安装三部曲

  • 1
    下载并拖入 /Applications
    首次打开若被 macOS 拦截，请在 Finder 中右键点击 Open。
  • 2
    授予权限
    系统设置 → 隐私与安全性 → 辅助功能，开启 Lexi 权限（用于读取选中文本）。
  • 3
    配置 API
    点击菜单栏图标 Settings，填入你的 Key。也可以先用 Google 引擎体验。
